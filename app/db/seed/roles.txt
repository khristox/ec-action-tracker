# Roles Configuration File for RMS - Action Tracker Focus
# Format: code|name|description|permissions
# Permissions are comma-separated, use * for all permissions
# 
# Permission Structure: resource:action
# Resources: meeting, minutes, action, participant, participant_list, document, dashboard, report, profile, settings
# Actions: create, read, update, delete, assign, complete, comment, upload, download, export, list, view, progress_update, my_tasks, overview

# ==================== SYSTEM ROLES ====================
admin|Admin|System Administrator|*

super_admin|Super Admin|Super Administrator with full system access|*

# ==================== MEETING MANAGEMENT ROLES ====================
meeting_creator|Meeting Creator|Can create and manage meetings|meeting:create,meeting:read,meeting:update,meeting:delete,meeting:list,minutes:create,minutes:read,minutes:update,minutes:delete,participant:create,participant:read,participant:update,participant:delete,participant_list:create,participant_list:read,participant_list:update,participant_list:delete,participant_list:use,document:upload,document:read,document:delete,dashboard:view
